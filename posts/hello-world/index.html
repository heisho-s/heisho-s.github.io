blog を作成してみた

2022年の Amazon の Black Friday Sale の際に、自前の Ubuntu の Native 環境が欲しいとふと思い、ミニ PC を購入した。
そしてせっかく環境が出来たので勉強がてら blog を作成してみた。

blog の構成

以下を使って静的ページをホスティングしている。

  • HUGO
    • 静的サイトジェネレータ
  • GitHub Pages
    • 静的ページのホスティング用
  • GitHub Actions
    • HUGO でビルドし生成した静的ページの自動デプロイ用

どれも初めて使用したが、電子の海の情報を頼りに特に詰まらず構築できた。
偉大な先人の記録に感謝。

blog のレイアウトについては HUGO の知識が少し必要なので徐々に改善していく予定。

blog に書く内容について

基本的に OpenJDK に関する内容を備忘録として残す予定。
なお OpenJDK に関しては初心者なので、基本的に内容のレベルは低くなると思われる。
